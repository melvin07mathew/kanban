COMPONENTS OF THE PROJECT

  1. User login(token based)
  2. Main Board with Lists
  3. List management
  4. Card management
  5. Summary page
  6. Board view with lists
  7. Ability to add or manage a list
  8. Ability to add or manage a card
  9. Ability to move cards between tasks/lists
  10. Ability to identify the completed tasks visually
  11. Ability to handle multiple language
  12. Create/Add a card to a list
  13. Edit: Change title or content
  14. Edit: Mark as complete
  15. Edit: Move the card from one list to another
  16. Remove/Delete a card
  17. Show the summary of cards per list basis
  18. No of completed tasks
  19. No of tasks crossed deadline
  20. Graph as to when tasks were completed

APIs for interaction with lists and cards

  1. CRUD on Lists
  2. CRUD on Cards

BACKEND JOBS

  1. Reminder of the deadline of the task one day before the deadline
  2. Monthly Progress Report
  3. User Triggered Async jobs - Exporting list and cards
  4. Caching list and cards with cache expiry
